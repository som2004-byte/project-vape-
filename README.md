# Vape Royale (Next.js + Tailwind)

Futuristic neon/royal vape storefront with 3D hover cards, floating accents, and bold typography. Built on Next.js 14 (App Router), React 18, and TailwindCSS.

## Quick start
```bash
npm install
npm run dev
# then open http://localhost:3000
```

- App Router (`app/page.tsx`) with React components for hero, categories, featured vapes, specs, reviews, and footer.

- Product data is the provided fictional list in `app/page.tsx`; swap when you have real SKUs or a backend.
