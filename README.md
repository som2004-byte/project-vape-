This is a minimal starter project configured with React, Vite, Tailwind and react-three/fiber placeholders.

## Quick start

1. Install dependencies:
   ```bash
   npm install
   ```

2. Run dev server:
   ```bash
   npm run dev
   ```

3. The app will run at http://localhost:5173 by default.

## Notes

- Your product images are included in `public/images/`.
- Add a real GLB model to `public/models/` and set `modelUrl` in `src/data.js` to enable 3D viewers.
- Optimize images for production (webp, compress).
